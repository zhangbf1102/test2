Troubleshooting common issues

If you encounter any issues with the EV-Charger, consult the troubleshooting section of the user manual. Common issues include faulty wiring, incorrect installation, and overheating.

  1. Check the charger and power source to make sure they are properly connected.
  2. Make sure the device is compatible with the charger and power source.
  3. Check the display for error messages and follow the instructions to resolve the issue.
  4. If the problem persists, contact customer support for further assistance.
